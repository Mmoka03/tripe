<h3>개발 시작하는 법</h3>

1. WSL2 Ubuntu 터미널을 킨다. (mac 유저는 별도)

2. git clone https://github.com/Mmoka03/tripe.git

3. cd tripe

4. sh setup.sh 실행 (비밀번호 1회 입력 필요)

5. 터미널 재시작 후 docker-compose up -d 입력
